# NeevAI - Immersive Edge-AI Healthcare Core

NeevAI is a futuristic, decentralized healthcare intelligence system utilizing high-performance edge computing, real-time computer vision analysis, and medical-audio diagnostic layers. It features an interactive, high-fidelity 3D central core driven by WebGL to visualize neural telemetry data seamlessly.

## 🚀 Live Demo
Deploy Link: https://neevai-peach.vercel.app

## 🛠️ Tech Stack & Architecture

- **Frontend Core:** React.js / Vite, TailwindCSS
- **3D Graphics Engine:** Three.js (WebGL)
- **Animation Pipeline:** GSAP (GreenSock Animation Platform) & ScrollTrigger
- **Build System & Package Manager:** Vite / Bun / Node.js
- **Hosting Platform:** Vercel

## ✨ Key Features

- **Interactive 3D Core:** A multi-layered, floating neural core built with mathematical geometries (Icosahedron/Torus) that shifts dynamically as users interact or navigate.
- **Cinematic Scroll-Flight:** Scroll-driven camera positioning that navigates the user smoothly through distinct technical layers (Vision, Audio, Edge Architecture).
- **Medical Edge Diagnostics:** High-speed processing models simulated natively on client environments for zero-latency feedback.
